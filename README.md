# GoodVoice

Голосовой ввод для Windows. Нажал клавишу — сказал — текст вставлен.

## Зачем это нужно

Windows имеет встроенный голосовой ввод (Win+H), но он:
- Отправляет голос в облако Microsoft
- Медленный и неточный
- Не работает оффлайн
- Неудобный интерфейс

**GoodVoice** делает то же самое, но:
- **Полностью оффлайн** — модель работает на вашем компьютере
- **Быстро** — обработка за секунды
- **Приватно** — голос никогда не покидает ваш компьютер
- **Красиво** — анимированный HUD показывает что происходит

## Как это работает

1. **Нажмите Right Ctrl** — появляется HUD, начинается запись
2. **Говорите** — бары пульсируют в ритм голоса, круг показывает громкость
3. **Отпустите Right Ctrl** — запись останавливается, текст распознаётся
4. **Текст вставляется** в активное поле (Блокнот, Word, браузер — куда угодно)

Всё. Никаких окон, никаких кнопок — просто говорите и текст появляется.

## Установка

```powershell
git clone https://github.com/Ap3x0s/goodvoice.git
cd goodvoice
pip install -r requirements.txt
```

## Запуск

```powershell
python main.py
```

При первом запуске модель скачивается автоматически (~1.5 ГБ). Далее работает оффлайн.

## Использование

| Действие | Результат |
|----------|-----------|
| **Right Ctrl (зажать)** | Начать запись |
| **Right Ctrl (отпустить)** | Вставить текст |
| **Right Ctrl (нажать)** | Toggle режим (опционально) |
| **Escape** | Отмена записи |
| **ПКМ по иконке в трее** | Настройки / Выход |

## Настройки

Файл: `~/Documents/goodvoice/settings.json`

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| `model_size` | `medium` | Размер модели: `tiny` (быстро) → `large-v3` (точно) |
| `language` | `auto` | Язык: `auto`, `ru`, `en`, `de`, `fr` и др. |
| `trigger_mode` | `hold` | `hold` (зажал-говоришь-отпустил) или `toggle` (нажал-говоришь-нажал) |
| `punctuation` | `true` | Автоматические запятые и точки |
| `hud_theme` | `hybrid_v2` | Тема интерфейса: `google`, `hybrid_v2`, `vercel` |
| `hotkey` | `ctrl_r` | Горячая клавиша: `ctrl_l`, `ctrl_r` |

## Темы HUD

| Тема | Стиль |
|------|-------|
| `google` | Мягкий, минималистичный |
| `google_v2` | Премиум с физикой пружин |
| `hybrid_v2` | ★ Основная — бары + таймер + круг громкости |
| `vercel` | Острый, кибер-стиль |

## Модели Whisper

| Модель | Размер | Скорость | Качество |
|--------|--------|----------|----------|
| `tiny` | 75 МБ | Быстро | Среднее |
| `base` | 150 МБ | Нормально | Хорошее |
| `small` | 500 МБ | Нормально | Отличное |
| `medium` | 1.5 ГБ | Медленно | Отличное |
| `large-v3` | 3 ГБ | Очень медленно | Максимальное |

Рекомендация: `medium` — лучший баланс скорость/качество.

## Структура проекта

```
goodvoice/
├── main.py              # Точка входа, связывает всё вместе
├── audio_recorder.py    # Запись звука с микрофона
├── transcriber.py       # Распознавание речи (Whisper)
├── text_inserter.py     # Вставка текста в активное окно
├── hotkey_manager.py    # Перехват горячих клавиш
├── hud_window.py        # Диспетчер тем HUD
├── hud_hybrid_v2.py     # Основная тема интерфейса
├── hud_google.py        # Тема Google/Catppuccin
├── hud_vercel.py        # Тема Vercel/Cyber
├── tray_icon.py         # Иконка в системном трее
├── settings.py          # Управление настройками
└── assets/              # Иконки
```

## Требования

- Windows 10/11
- Python 3.10+
- Микрофон

## Приватность

- Голос обрабатывается **локально** на вашем компьютере
- Никакие данные не отправляются в интернет
- Модель скачивается один раз при первом запуске
- История и настройки хранятся только у вас: `~/Documents/goodvoice/`
